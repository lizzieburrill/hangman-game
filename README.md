# React Hangman Capstone

A classic Hangman game built with React.

## How to Play
1. The game selects a random word.
2. Guess the word by clicking letters on the onscreen keyboard.
3. Correct guesses fill in the blanks.
4. Incorrect guesses add a part to the Hangman drawing.
5. You have 10 mistakes before the game ends!

## Installation & Running
1. Open your terminal.
2. Navigate to this folder.
3. Install dependencies:
   `npm install`
4. Start the game:
   `npm start`
5. Open [http://localhost:3000](http://localhost:3000) to play.

## Technologies Used
- React (Create React App)
- CSS for styling
- SVG for dynamic drawing
